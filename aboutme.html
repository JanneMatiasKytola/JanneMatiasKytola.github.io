First game I made that didn't stay to be small prototype testing. Even though this game wasn't meant to be published and was made mainly with learning purposes, it has ended up being in here in my collection of shit games I've made.

WARNING: If you choose to play game with audio on, better lower volume before pressing top right corner audio button.

Made with Macromedia flash 8 and actionscript2. Art assets are made by me, and there you can see on menu screen, attempt to mix pixel art with another art style. I've made some elements to menu to make it more athmospheric, like fog and rain, and smooth character sprites. If you look closely you can see light flickkering on background tower highest room. This ain't good piece of art, and it has a lot of flaws - like that unnatural shield movement and background pixel art piece lost quality over the progress.

Making games with flash can be FAST, since drawing and animating tools are inside the flash software! To add code to animation characters, just need to select character and correct keyframe and add code to the character object, so there isn't burning time on inserting images on screen and moving image files into the game engine images file folder. Rip flash though.
